# Texsec local run

## Requirements

- Node.js 18 or newer
- npm

## First install

```bash
npm install
npm run install:all
```

## Run app locally

```bash
npm run dev
```

Open the client at:

```text
http://127.0.0.1:5173
```

The API server runs at:

```text
http://127.0.0.1:3001
```

Health check:

```text
http://127.0.0.1:3001/api/health
```

Admin login:

```text
admin / admin123
```

## Run only the API server

```bash
npm run server
```
